## Technologies
Project is created with:
* Flask
* Angular
	
## Setup Backend
To run the backend server:

1. Go to backend folder
```
$ pip3 install flask flask_cors
$ python3 app.py
```

## Serve the Frontend

To run the frontend:

### If you already have the Angular installed 

```
$ npm install
$ ng serve
```
### else

```
$ npm install -g @angular/cli
$ cd ../path_to_frontend
$ npm install
$ ng serve
```
